# 🚀 Vetra - פלטפורמת אוטומציית שיווק AI

<img src="https://github.com/user-attachments/assets/df541304-ea1c-4ebe-b6fd-2e093680314f" alt="Vetra Thumbnail">

## 🌟 מבוא
Vetra היא פלטפורמת אוטומציית שיווק מבוססת AI שמשנה את תהליך השיווק שלך תוך שניות. נבנתה עם טכנולוגיות מודרניות, היא מציעה ממשק מלוטש לניהול קמפיינים שיווקיים, יצירת תוכן ואנליטיקת ביצועים.

## 💻 Tech Stack

- **Next.js 15** – React framework for production
- **TailwindCSS** – Utility-first CSS framework
- **Shadcn UI** – Reusable component system
- **Framer Motion** – Smooth animations
- **TypeScript** – Type-safe code
- **Number Flow** – Smooth number animations

## 🛠️ Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/bforbilly24/vetra-nextjs-frontend-app.git
    ```

2. Install dependencies:
    ```bash
    pnpm install
    # or
    yarn install
    ```

3. Run the development server:
    ```bash
    pnpm run dev
    # or
    yarn dev
    ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 🔧 Environment Variables

Create a `.env` file in the root directory:

```env
NEXT_PUBLIC_APP_URL=your_app_url
```
